# MindPolis Features

## 1. Dynamic Assessment Engine
- Support for complex instrument routing (Political Compass, Moral Foundations Questionnaire).
- Continuous scoring calculation using real-time dimension weighting (1-5 confidence sliders).
- Resumable database-backed sessions.

## 2. The Result Experience
- **Ideological Radar Map:** 8-axis (or 6-axis) multidimensional plotting.
- **Population Distribution:** Overlay user score against generalized normative data.
- **Cognitive Archetypes:** Map mathematical clusters to accessible personas (e.g., "The Civic Rationalist").

## 3. Academic Explanation Layer (AEL)
- **Progressive Disclosure:** 1-sentence summaries backed by expandable, heavily cited methodology deep-dives.
- Integrates definitions from Moral Foundations Theory, Schwartz Theory of Basic Values, and World Values Survey data.

## 4. Gamification: "Cognitive Signifiers"
- Minimalist, rarity-based badge system (Common to Anomaly).
- Rewards given for *latency* (The Deliberator) and *consistency* (The Architect), not ideological extremity.

## 5. Viral Infrastructure
- Auto-generated cryptographic share-hashes (`/r/[hash]`).
- Pre-filled semantic sharing templates for X/Twitter.
- Programmatic Open Graph (OG) image generation for rich-link rendering.

## 6. Political Figure Similarity (Philosophical Resonance)
- Cosine-similarity matching against expert-coded historical vectors.
- Strict built-in friction mechanics (high disagreement in a core trait removes the figure from results).
- "No-Monsters" protocol.

## 7. Administrative Control Panel
- Role-based Access Control (RBAC) protecting `/admin` routes.
- Real-time logging of submissions, anonymized IPs, and aggregated dimension trends.
